# Tic Tac Toe (C++)
A simple Tic Tac Toe game written in C++.

## How to Play
1. The game is played on a 3x3 grid.
2. Player 1 can choose either **X or O** and Player 2 is other than what player 1 chose.
3. Players take turns entering the number of the slot (1-9) where they want to place their marker.
